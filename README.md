# 三防形势智能研判与指挥辅助系统 MVP v1.0

## 🎯 系统简介

**一套基于 GIS 地图 + 地形高程 + 多级气象 + AI 智能研判的三防辅助决策系统**

### 核心能力
✅ 自由圈选任意区域（区/街镇/自定义片区）  
✅ 真实地形图 + 高程数据，判断哪里会淹  
✅ 五级气象数据自动抓取（中央+省市区+民间）  
✅ AI 智能研判：风险等级 + 危险点位 + 指挥建议  
✅ 一键生成研判简报，直接可用

## ⚙️ 配置说明

### 1. 高德地图 KEY 配置（重要）
打开 `index.html`，找到第6行：
```html
<script src="https://webapi.amap.com/maps?v=2.0&key=YOUR_AMAP_KEY"></script>
```
将 `YOUR_AMAP_KEY` 替换为你的高德地图密钥。

**获取方式：**
1. 注册高德开放平台：https://lbs.amap.com/
2. 创建应用 → 添加Key（Web端）
3. 复制Key替换到代码中

### 2. 气象数据源配置
编辑 `crawler.py`，修改第12行：
```python
TARGET_AREA = "广州市"  # 改成你的目标区域
```

### 3. 地形API配置（可选）
`terrain.py` 使用公开免费的高程API，无需配置。  
如需更高精度，可替换为Google Elevation API。

## 🗺️ 地形分析说明

### 核心原理
**地形高程 + 降雨量 = 淹没风险**

系统通过以下维度判断：
1. **高程数据**：识别低洼区（<50m）、极低洼区（<30m）
2. **地理位置**：沿河、隧道、地下空间
3. **降雨预报**：24小时累计雨量
4. **历史记录**：曾淹水点位优先级提升

### 风险计算公式
```
风险得分 = 高程风险（30分）+ 降雨风险（25分）
          + 地理位置（20分）+ 特殊空间（25分）
```

### 输出结果
- **风险等级**：低风险/中风险/高风险/极高风险
- **淹没预判**：积水深度、淹没面积、持续时间
- **应对建议**：封闭/巡查/转移/布防

## 📊 数据说明

## ❓ 常见问题

### Q1：地图不显示怎么办？
**A**：检查高德地图KEY是否配置正确。

### Q2：气象数据抓不到？
**A**：
1. 检查网络连接
2. 目标气象网站可能改版，需更新爬虫
3. 暂时使用模拟数据测试

### Q3：地形高程不准？
**A**：公开DEM精度30-90米，适用于判断大致地势，不做精密测量。

### Q4：能用手机吗？
**A**：网页支持手机访问，但推荐电脑端使用（地图操作更方便）。

### Q5：如何添加本地隐患点？
**A**：编辑 `ai_judge.py`，在测试用例部分添加数据，或对接本地数据库。
